# Surovi podatki

Ta mapa vsebuje originalne podatke, kot so bili preneseni iz vira.

Pravilo:
- tukaj ničesar ročno ne popravljamo
- vsak vir ima svojo podmapo
- vsaka podmapa vira naj ima `README.md` z URL-jem vira, datumom prevzema in kratkim opisom vsebine
- vse čiščenje se izvaja šele v `pipelines/clean/` ali `scripts/data/`
