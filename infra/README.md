# Infrastruktura

`infra/` vsebuje tehnične opombe za poganjanje in objavo projekta.

Trenutni live deployment uporablja statični Vercel build, konfiguriran v
[`vercel.json`](/Users/zankespert/Desktop/KlopPodKlopjo/vercel.json).

- `docker/`: prostor za morebitne prihodnje Docker datoteke
- `deploy/`: prostor za dodatne deployment manifeste ali zapiske

Aktualni postopek za live deploy je opisan v
[docs/live-deployment.md](/Users/zankespert/Desktop/KlopPodKlopjo/docs/live-deployment.md).
